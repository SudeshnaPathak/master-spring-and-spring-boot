ECMAScript -> Standard
JavaScript -> Implementation

With the Evolution of ECMAScript versions ~ ES7 , ES8 , .....ES14 writing javascript code is not so difficult anymore
To run javascript code we need nodejs and npm

npm ~ Package manager for javacript similar to maven/Gradle
npx ~ Package executer, Execute javascript packages w/o installing
npm init : Create a new project
package.json : Dependency definitions
npm install <dependency-name> : To Add a dependency(npm automatically downloads the dependencies)
node_modules : Dependencies are downloaded to node_modules ~ temp folder which stores all our downloaded dependencies
